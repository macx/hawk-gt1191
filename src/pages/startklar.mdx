## Warum die Basics den Unterschied machen

KI-Tools wie ChatGPT oder Copilot generieren Code in Sekunden. Ohne Grundlagen kannst du ihn weder beurteilen noch verbessern. Jede Website beruht auf denselben drei Sprachen: **HTML** für Struktur, **CSS** für Gestaltung, **JavaScript** für Interaktion. Wer sie versteht, baut robuster, schneller und zugänglicher – und weiß, wie man KI gezielt einsetzt.

### HTML

<abbr title='Hypertext Markup Language'>HTML</abbr>[^1] ist die
Auszeichnungssprache, mit der Struktur und Inhalt einer Webseite aufgebaut
werden – von Überschriften über Links bis hin zu Bildern und Formularen.

<Figure
  caption='Ein `h1`-Element markiert die wichtigste Überschrift auf der Seite und
    gibt ihr gleichzeitig eine Bedeutung – für Browser, Suchmaschinen und
    Screenreader.'
>

```html title="index.html"
<h1>Das ist ein HTML-Element</h1>
```

</Figure>

### CSS

<abbr title='Cascading Stylesheets'>CSS</abbr>[^3] ist eine Sprache, die
verwendet wird, um das Aussehen einer Webseite zu definieren. Mit CSS können
Farben, Schriftarten, Layouts und andere visuelle Aspekte einer Webseite
angepasst werden.

<Figure
  caption='Eine CSS-Regel besteht aus Selektor und Deklaration: Welches Element soll
    wie aussehen? Dieses Beispiel färbt alle `h1`-Elemente rot.'
>

```css
h1 {
  color: red;
}
```

</Figure>

### JavaScript

JavaScript ist eine Skriptsprache, die verwendet wird, um die Funktionalität einer Webseite zu erweitern. Mit ihr können dynamische Inhalte, interaktive Elemente und Benutzerinteraktionen erstellt werden.

### Studienstruktur in der Praxis

So kannst du dein Studium sinnvoll aufbauen, ohne dich zu verzetteln:

1. Mit dem Lernpfad [Einstieg](/tutorials/lernpfade/einfuehrung) starten:
   Du arbeitest dort klassisch mit HTML- und CSS-Dateien, damit du beides von
   Grund auf lernst.
2. Danach das [Starterkit](/startklar/starterkit) über die Kommandozeile installieren
   und einrichten. Es legt dir die Projektstruktur an und bringt die wichtigsten
   Features mit, die du für die Webentwicklung brauchst.
3. Als Nächstes den Lernpfad [Kernkonzepte](/tutorials/lernpfade/grundlagen)
   durcharbeiten. Dort lernst du Layout-Techniken, semantische Strukturierung
   und weitere Basisprinzipien fürs Web.
4. Nun widmest du dich wieder den kuratierten Tutorials im Lernpfad. Weiter geht es mit
   [UI & Gestaltung](/tutorials/lernpfade/gestaltung), das auf Einstieg und
   Kernkonzepten aufbaut.
5. Die Seiten [Assets](/assets) und [Dokumentation](/dokumentation) früh und
   regelmäßig nutzen. Assets helfen dir beim Bauen, Dokumentationen vertiefen
   das Verständnis von HTML und CSS neben den Tutorials.
6. Zum Abschluss führt dich [Abgabe und Anforderungen](/startklar/abgabe) durch die
   komplette Checkliste, damit du nichts vergisst.

[^1]: HTML steht für HyperText Markup Language und ist eine Auszeichnungssprache, die verwendet wird, um Webseiten zu erstellen. HTML-Dokumente bestehen aus einer Folge von Tags, die die Struktur und den Inhalt einer Webseite beschreiben.

[^2]: Ein Screenreader ist eine Software, die den Inhalt einer Webseite in Sprache umwandelt und so blinden oder sehbehinderten Menschen den Zugang zu digitalen Inhalten ermöglicht.

[^3]: CSS ist eine Stylesheet-Sprache, die verwendet wird, um das Aussehen einer Website zu definieren. Sie definiert die Farben, Schriftarten, Layouts und andere visuelle Aspekte einer Website.

[^4]: Sass steht für „Syntactically Awesome Style Sheets“ und ist eine CSS-Erweiterungssprache. Es handelt sich um eine mächtige und flexible Stylesheet-Sprache, die es Entwicklern ermöglicht, effizienter und produktiver CSS-Code zu schreiben.
